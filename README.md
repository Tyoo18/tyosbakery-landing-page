# 🥐 TyosBakery Landing Page

A modern and playful French-inspired bakery landing page created for portfolio purposes.

TyosBakery is a fictional trendy patisserie brand focused on creating an elegant, Instagrammable, and cozy digital experience with soft pastel aesthetics and interactive UI elements.

---

## ✨ Preview

TyosBakery combines:
- French patisserie aesthetics
- Editorial-style typography
- Playful pastel colors
- Smooth scroll animations
- Interactive product showcases
- Responsive modern layouts

---

## 🚀 Features

- Sticky Navbar
- Hero Section with Floating Cards
- Signature Pastries Showcase
- Best Seller Highlight
- Interactive Menu Preview
- Testimonial Slider (Swiper.js)
- Instagram Gallery
- Google Maps Embed
- Floating WhatsApp Button
- Smooth Scroll Animations (AOS)
- Fully Responsive Layout
